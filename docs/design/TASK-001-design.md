## 1. 设计思路

### 1.1 设计理念
- **简洁直观**: 一眼看清团队状态
- **卡片式布局**: 每个成员/任务独立卡片
- **实时反馈**: 状态变化即时可见
- **温暖风格**: 配合虾管事🤍的温暖可靠 vibe

### 1.2 配色方案
| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色调 | 温暖白 | `#F5F5F5` |
| 强调色 | 爱心红 | `#FF6B6B` |
| 成功状态 | 绿色 | `#51CF66` |
| 警告状态 | 黄色 | `#FFD43B` |
| 危险状态 | 红色 | `#FF6B6B` |
| 文字主色 | 深灰 | `#343A40` |
| 文字次要 | 浅灰 | `#868E96` |

## 4. 交互设计

### 4.1 悬停效果
- 成员卡片悬停：轻微放大 + 阴影加深
- 任务卡片悬停：背景色变浅

### 4.2 点击交互
- 点击成员卡片 → 显示成员详情
- 点击任务卡片 → 显示任务详情
- 点击 [+ 新建任务] → 弹出创建表单

### 4.3 刷新机制
- 页面每 30 秒自动刷新数据
- 下拉手动刷新
